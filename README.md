# POMODORO-JS

## Description

Minimalistic, 0 dependency terminal concentration tool implementing pomodoro technique.

## Installation

### Prerequisites

- Node.js >= 17.5.0

### Usage

Run `npx cli-pomodoro-js` in your terminal.

### Controls

- `r` - start the timer
- `s` - skip the current phase
- `p` - pause/resume the timer
- `q` - quit the timer
